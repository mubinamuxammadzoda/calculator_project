Loyiha haqida
Ushbu ilmiy kalkulyator PHP dasturlash tilida yaratilgan bo'lib, matematika fanini o'qitishda shaxsga yo'naltirilgan ta'lim metodlarini qo'llashga mo'ljallangan. Dastur talabalar va o'quvchilarga murakkab matematik amallarni tezda va aniq bajarishga yordam beradi. Frontend HTML, CSS va JavaScript yordamida, backend esa PHP orqali amalga oshirilgan. Kunduzgi va tungi ko'rinish, hisoblash tarixi va xotira funksiyalari ham qo'shilgan.
⚙️
PHP Funksiyalari
  • safe_eval() Matematik ifodani xavfsiz hisoblash
  • sin / cos / tan Trigonometrik funksiyalar
  • log10() O'nlik logarifm (log)
  • log() Natural logarifm (ln)
  • sqrt() Kvadrat ildiz
  • pow() Daraja hisoblash (x², x³, xʸ)
  • factorial Faktorial hisoblash (n!)
  • deg2rad() Darajani radianga aylantirish
  • abs() Mutlaq qiymat
  • calculate_action() Amallarni boshqaruvchi funksiya
🧮
Kalkulyator imkoniyatlari
  • Arifmetika Qo'shish, ayirish, ko'paytirish, bo'lish, qavslar
  • Trigon. sin, cos, tan — daraja yoki radian rejimida
  • Logarifm log₁₀ va tabiiy logarifm ln
  • Darajalar x², x³, ixtiyoriy daraja xʸ, kvadrat ildiz √x
  • Maxsus π, e, n!, 1/x, foiz (%)
  • Xotira MS, MR, M+, M−, MC
  • Tarix So'nggi 20 ta hisob-kitob
  • Klaviatura To'liq klaviatura qo'llab-quvvatlash
  • Tema Kunduzgi va tungi ko'rinish
  • Fallback PHP yo'q bo'lsa JS da hisoblash

Foydalanilgan texnologiyalar
PHP 8 HTML5 CSS3 JavaScript ES6 REST API (JSON) Google Fonts CSS Variables
Backend calculator.php POST so'rovlarni qabul qilib, JSON formatida javob qaytaradi. Frontend JavaScript orqali real vaqtda serverga murojaat qiladi. Agar PHP server mavjud bo'lmasa, jsEval() va jsFn() funksiyalari zaxira sifatida ishlaydi.
🏗️
Dastur arxitekturasi
index.html — Foydalanuvchi interfeysi: navigatsiya paneli, kalkulyator tugmalari, displey, tarix paneli va kunduz/tun rejim almashtirgichi. Barcha UI logikasi shu faylda.

calculator.php — Server tomoni: matematik ifodalarni eval() orqali xavfsiz hisoblaydi, trigonometrik, logarifmik va boshqa funksiyalarni qayta ishlaydi, xatolarni ushlab oladi va JSON javob yuboradi. Faqat ruxsat etilgan belgilar qabul qilinadi — xavfsizlik ta'minlangan.
